Challenge OdontoPrev - API

Sobre o Projeto

Esta é uma API para gestão odontológica, permitindo que pacientes agendem consultas, avaliem clínicas odontológicas e acessem serviços oferecidos.
A aplicação está hospedada na Azure utilizando Docker e Azure Container Instances (ACI) e conecta-se ao banco de dados Oracle da FIAP.

 Tecnologias Utilizadas

Spring Boot (Backend)

Oracle Database (FIAP)

Docker

Azure Container Instances (ACI)

Maven
